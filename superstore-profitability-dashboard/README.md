# Superstore Profitability Analysis Dashboard (Power BI)

## Project Overview
This project analyzes business profitability using the Superstore dataset.  
The goal is to move beyond sales numbers and understand where the business is generating profit and where it is losing money.

Using Power BI, this dashboard explores profit performance across regions, categories, and products.

---

## Key Metrics

Total Orders: 5K  
Total Profit: 286K  
Profit Margin: 12.5%

---

## Dashboard Features

- KPI cards showing total orders, profit, and profit margin
- Regional profit performance analysis
- Category profit contribution
- Top profitable products
- Loss-making sub-categories using conditional formatting
- Interactive region slicer for filtering

---

## Key Insights

- Technology category generates the highest profit
- West region contributes the most profit
- Some sub-categories like Tables and Bookcases generate negative profit
- Overall business profit margin is 12.5%
